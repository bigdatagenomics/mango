Mango User Guide
================

Introduction
============

Mango is a distributed visualization tool that enables visualization of genomic data on top of `Apache
Spark <https://spark.apache.org>`__.


The Mango/Big Data Genomics Ecosystem
-------------------------------------

Mango builds upon the open source `Apache
Spark <https://spark.apache.org>`__, `Apache
Avro <https://avro.apache.org>`__, and `Apache
Parquet <https://parquet.apache.org>`__ projects. Additionally, Mango can
be deployed for both interactive and production workflows using a
variety of platforms.
